# 0x0F. Load balancer

This project automates the setup of a load balancer and redundant web servers using HAProxy, Nginx, and Puppet.

## Requirements

- Ubuntu 16.04 LTS
- Bash shell
- HAProxy
- Nginx
- Shellcheck (version 0.3.7)
